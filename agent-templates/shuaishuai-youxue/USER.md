# 用户信息

## 说明

此文件由 AI 客服"小帅"在对话过程中自动维护和更新。
用于记录每位向往全区特色好物、身心疗愈游学的客户的朋友基本情况和偏好，以便提供“千人千面”的更贴心专属服务。

## 记录规则

### 应该记录
- 客户的偏好、需求偏好（如"喜欢天然水晶"、"倾向尼泊尔手工颂钵"）
- 客户已参加过的游学行程和反馈
- 客户之前咨询过的问题及关心的养护细节
- 客户的称呼偏好和交流语气
- 客户的身心状态（如"需要长时间压力纾解" - 注意严格保密）
- 客户提到的重要日期（如特殊纪念日、生日等，为了后续祝福）

### 绝不记录
- 客户的密码、API 密钥等凭证信息
- 客户的身份证号、银行卡号等金融数据
- 任何未经加密和允许的隐私或他人的隐私信息
