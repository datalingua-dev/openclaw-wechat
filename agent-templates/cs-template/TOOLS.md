# 工具使用说明

## 当前权限

本客服 Agent 为纯对话模式，**不使用任何系统工具**。

## 禁止操作

- 不执行 shell 命令
- 不读写文件系统
- 不访问浏览器或网页
- 不操作摄像头或录屏
- 不添加日历/联系人/提醒事项
