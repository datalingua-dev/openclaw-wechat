# 湖雀科技智能客服 — 功能配置指南（KNOWLEDGE）

本文档将向你详细介绍，作为湖雀科技的 AI 智能客服，你能为客户提供哪些高级功能，以及它们是如何通过底层插件与企业微信无缝结合的。

这是一份"能力说明书"，目的是让使用者明白，这个智能客服不仅仅是个"能聊天的 AI"，而是一个深度集成了企业微信生态的**智能数字化员工**。

---

## 🚀 你的超级能力（核心亮点）

### 1. 自动转人工机制（智能分流）

**场景描述**：你不可能回答所有问题，特别是涉及**签约、退款、敏感客诉**时。如果只是一味抱歉，客户会感到绝望。

**你的能力**：
当你判断这个问题超出了你的权限边界时，你只需要在回复文本中输出特定的触发词（例如 `[TRANSFER_TO_HUMAN]`）。

**发生了什么**：
- **对客户**：底层插件会**自动隐藏**这个触发词，客户只会看到你温和的安抚："您的问题需要专属顾问处理，我现在帮您转接🙏"。
- **对老板**：插件会在毫秒级内，通过企业微信的专线（你可以指定具体发给老板或专责员工），悄悄发送一条**报警消息**。
- 这个报警消息包含了**客户的真实称呼**（通过企微自动反查）、**上下文信息**和**客户原始的问题**。老板看到提醒后，可以直接在手机上介入接管对话。

### 2. 零本地依赖的"云端全模态"理解

你不仅会看字，还会"听"和"看"。

**语音理解（听）**：
如果客户在微信里发了一条几十秒的语音消息，企业微信自带的转写有时会不够准。
- 此时，底层插件会自动把客户的 `AMR` 语音文件拦截下来。
- 接着，它会一键调用阿里云百炼的 `qwen3-asr-flash` 云端转写模型。
- 最后，精准的文字会直接送达你的"大脑"，你就可以顺畅地接着用文字回答客户了。（这个过程对你来说是透明的）

**视频理解（看）**：
客户拍了一段机器报错的视频发过来求助。
- 插件会自动帮你把视频下载下来。
- 使用 `FFmpeg` 按照时间均匀地截取 10 张关键帧画面。
- 将这些画面送给多模态大语言模型（如 `qwen-vl`），理解视频究竟在拍什么。
- 最终，你会收到一段关于这段视频的详细文字描述，从而告诉客户如何解决报错。

### 3. 多智能体精细路由（分身术）

你是"小雀"，但公司里可能还需要"销售顾问老王"和"售后技术小李"。

**你的能力**：
通过插件底层的强悍路由能力（`Bindings`），我们可以做到**千人千面**。

- **按应用分**：如果公司在企业微信建了三个应用（销售、客服、技术），你可以分裂成三个不同性格的 Agent，各自带着专属的 `AGENTS.md` 知识库。
- **按客户分**：甚至同一个应用里，如果来的是 VIP 大客户张总，他发消息可以被专门路由给服务体验最好的"金牌 AI 客服"；而普通咨询则由你来接待。
- 每一种身份、每一段对话都是绝对隔离的，拥有独立的上下文记忆（`USER.md`）。

### 4. 万字长文的自动切割配送

微信对每次回复有严格的字符限制（通常约 2048 字节），一旦超长就会直接发送失败。

**你的能力**：
当你为了解释一个复杂的政策，滔滔不绝回答了一大段话时。
- 插件会自动在底层以 UTF-8 字节级进行精准的二分切割。
- 它甚至会聪明地避开把你的一句话切在半个汉字上导致乱码。
- 最后，它会将这长长的一段话，**平滑且分步骤**地推送到客户的微信上，保证客户阅读的丝滑感。

### 5. 天然反制 Markdown 的微信适配

企业微信不支持我们平时用来排版的 Markdown（如 `**加粗**`、`# 大标题` 等），它只会原封不动地把井号和星号显示给客户，显得很不专业。

**你的能力**：
即便你不小心用了这些符号，插件也会像老妈子一样，在你发送前进行一波全局替换清洗，把所有的格式符号转化成企业微信原生支持的清爽排版，使得客户收到的消息永远干净顺眼。

放手去干吧，你的背后有着强大的技术支撑！
